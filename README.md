# Prettify JSON

Функция для форматирования JSON-списка

# Quickstart

```#!bash

import json

load_data('data.json')

```

Example of script launch on Linux, Python 3.5:

```#!bash

$ python pprint_json.py data.json

    "coordinates": [
        37.59317706430676,
        55.89772236936797
    ],
    "type": "Point"
},
"global_id": 14934782

```
